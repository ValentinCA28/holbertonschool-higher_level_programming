This module defines a function ``text_indentation(text)``.

Importing the function:
    >>> text_indentation = __import__('5-text_indentation').text_indentation

Basic usage
===========

    >>> text_indentation("Hello world. How are you?")
    Hello world.
    <BLANKLINE>
    How are you?
    <BLANKLINE>

Multiple punctuation
====================

    >>> text_indentation("First line: second line. Third line?")
    First line:
    <BLANKLINE>
    second line.
    <BLANKLINE>
    Third line?
    <BLANKLINE>

Leading and trailing spaces
===========================

    >>> text_indentation("  Space before.  Space after? ")
    Space before.
    <BLANKLINE>
    Space after?
    <BLANKLINE>

Empty string
============

    >>> text_indentation("")

TypeError if not string
=======================

    >>> text_indentation(123)

    ...

    >>> text_indentation(None)

Multiple lines, multiple punctuations
=====================================

    >>> text_indentation("Lorem ipsum. Quonam modo? Non autem hoc: Plus semper voluptatis?")
    Lorem ipsum.
    <BLANKLINE>
    Quonam modo?
    <BLANKLINE>
    Non autem hoc:
    <BLANKLINE>
    Plus semper voluptatis?
    <BLANKLINE>

Mixed spacing and punctuation
=============================

    >>> text_indentation(" A. B? C: D")
    A.
    <BLANKLINE>
    B?
    <BLANKLINE>
    C:
    <BLANKLINE>
    D

Single word, no punctuation
===========================

    >>> text_indentation("Holberton School")
    Holberton School

Trailing punctuation only
=========================

    >>> text_indentation("End here:")
    End here:
    <BLANKLINE>
